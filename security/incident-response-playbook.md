# Incident Response Playbook (Reference)

1) Triage
- Classify severity
- Identify impacted assets

2) Containment
- Isolate host/network segment
- Revoke/rotate credentials

3) Eradication
- Remove persistence
- Patch vulnerabilities

4) Recovery
- Restore services
- Monitor for reoccurrence

5) Post-incident
- Root cause analysis
- Lessons learned and backlog actions
