System Engineer Foundation
===========================

Linux, Windows, Networking, Security, Automation, Kubernetes, Monitoring


Task:

1. create a linux and windows server on aws.
2. create a ansible inventory file to manage the servers.
3. create a ansible playbook to install nginx on linux server and iis on windows server.
4. create a ansible playbook to configure firewall on linux and windows server.
5. create a ansible playbook to deploy a sample web application on linux and windows server.
6. create a ansible playbook to monitor the servers using nagios.
7. create a ansible playbook to backup the servers using rsync.
8. create a ansible playbook to update the servers using yum and windows update.
9. create a ansible playbook to secure the servers using selinux and windows firewall.
10. create a ansible playbook to automate the deployment of a kubernetes cluster on aws.
